VIT Bhopal Health Centre - Appointment System

The Problem:
Booking doctor appointments at campus health centre is challenging - long queues, unclear availability, and manual processes cause delays for students and staff.

The Solution:
A Python-based appointment system that simplifies booking medical consultations through an easy-to-use console interface.

What It Does:
• Displays medical departments with consultation fees
• Collects and validates student information
• Allows booking for future dates with time slots
• Generates confirmed appointments with unique IDs
• Provides overview of all bookings

Key Features:
• Simple menu navigation
• Input validation and error handling
• Date validation (no past dates)
• Instant booking confirmation
• View all appointments

Technical:
• Built with Python
• Uses dictionaries, lists, and functions
• datetime module for date handling
• Console-based application

Perfect for students to quickly book appointments and health centre staff to manage schedules efficiently.

A practical campus solution making healthcare access easier!
