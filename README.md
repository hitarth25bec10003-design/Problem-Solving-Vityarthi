VIT Bhopal Health Centre Appointment System

A simple command-line appointment booking system for VIT Bhopal Health Centre. This system allows users to easily book appointments by selecting departments, time slots, and dates. It also provides options to view booked appointments and check department consultation fees.

Features
	•	Book appointments with department and time slot selection
	•	Select preferred date for appointments
	•	View all booked appointments
	•	Display available department fees
	•	Simple and user-friendly command-line interface

 Usage
	1.	Run the script in your terminal or command prompt.
	2.	Follow the on-screen prompts:
	•	Select department
	•	Choose date
	•	Pick time slot
	3.	Confirm your appointment.
	4.	You can also:
	•	View all booked appointments
	•	Check department fees

 How It Works
	•	The system displays available departments.
	•	Users choose a department and time slot.
	•	Appointment details are stored in the system.
	•	Users can view all appointments anytime.

Purpose

This project aims to simplify the appointment booking process for the VIT Bhopal Health Centre using a lightweight and easy-to-use command-line system.

Technologies

•	Python 3
